# Basic Rule-Based Chatbot in Python

## Project Description
This project is a simple rule-based chatbot created using Python. It can respond to common user questions and interact in a friendly conversational style.

## Features
- Responds to greetings
- Answers common questions
- Gives motivational replies
- Fun / casual responses
- Emoji-based friendly chat
- Ends conversation when user says bye/see you/good night

## Technology Used
- Python Programming Language

## How to Run the Program
1. Install Python on your system.
2. Save the code in a file named ` codeAlpha_chatbot.py`.
3. Open the terminal/command prompt.
4. Navigate to the folder where the file is saved.
5. Run the following command:
	python codeAlpha_chatbot.py
