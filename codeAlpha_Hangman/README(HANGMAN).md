# Hangman Game (Beginner Python Project)

## Project Goal
Create a simple text-based Hangman game using Python where the user guesses a word one letter at a time.

## Features
- Uses a fixed list of 5 predefined words  
- Allows 6 incorrect guesses  
- Console-based (no graphics)  
- Beginner-friendly code using lists, loops, and if-else

## Key Concepts Used
- `random` module  
- `while` loop  
- `if / else`  
- Strings and lists  
- User input and printing output

##  How to Run
1. Install Python (if not already installed).  
2. Save the code in a file named `hangman.py`.  
3. Open command prompt or terminal.  
4. Navigate to the folder containing the file.  
5. Run:
          python codeAlpha_hangman.py
